# План: единая точка оформления — корзина и чат-бот

## Что меняем

### 1. Убираем «Быстрый заказ» (иконка-молния) с фото товаров
- **`src/components/ProductCard.tsx`** — удалить кнопку с иконкой `Zap`, состояние `quickOrder`, импорт `QuickOrderModal` и его рендер. На карточке остаётся только иконка корзины.
- **`src/pages/ProductPage.tsx`** — удалить кнопку «Быстрый заказ», состояние `quickOrder`, импорт и рендер `QuickOrderModal`. На странице товара остаётся только «В корзину».
- **`src/components/QuickOrderModal.tsx`** — удалить файл целиком.

### 2. Заявки на декор — только через чат-бота
- **`src/components/DecorCard.tsx`** — кнопка «Оставить заявку» больше не открывает модалку. Вместо этого она программно открывает плавающий чат-виджет (через общий стейт) и подставляет в первое сообщение название услуги: «Здравствуйте! Хочу оставить заявку на услугу: {service.name}».
- **`src/components/DecorRequestModal.tsx`** — удалить файл целиком.
- **`src/components/admin/DecorRequestsTab.tsx`** и связанный стейт `decorRequests` в `StoreContext` — оставляем как есть (заявки, прилетевшие ранее, всё ещё видны в админке; новые заявки теперь будут приходить через чаты в `ChatsTab`).

### 3. Открытие чата извне
Сейчас `ChatWidget` управляет своим `isOpen` локально. Чтобы карточка декора могла его открыть, добавим лёгкий механизм:
- Расширить `StoreContext` полями `chatOpen: boolean`, `chatPrefill: string | null`, `openChat(prefill?: string)`, `closeChat()`.
- `ChatWidget` будет читать `chatOpen` из контекста (вместо локального стейта) и подставлять `chatPrefill` как начальное значение в поле ввода первого сообщения после регистрации.
- `DecorCard` вызывает `openChat("Здравствуйте! Хочу оставить заявку на услугу: …")`.

### 4. Что НЕ трогаем (по согласованию)
- Корзина (`CartDrawer`) и её форма оформления — остаются.
- Телефон в футере — остаётся.
- Поле телефона в форме регистрации чата — остаётся (бот должен знать, кому перезвонить).
- Админка: вкладки «Заказы», «Чаты», «Заявки на декор» — остаются.

## Ожидаемый результат

```text
Карточка товара ──► [В корзину] ──► Корзина ──► Оформить заказ
Карточка декора ──► [Оставить заявку] ──► Чат-бот (с подставленным текстом)
Любой вопрос    ──► Плавающий чат справа снизу
```

Все «быстрые» пути оформления теперь идут через чат-бота, а покупка товаров — через корзину. Никаких отдельных модалок с телефоном под фото больше нет.
