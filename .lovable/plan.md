

## План: Luxury-доработка сайта "Магия Роз"

Усиление premium-ощущения через тексты, визуал и trust-элементы. Структура остаётся, меняются только стиль и контент.

---

### 1. HeroSection.tsx — новые тексты + trust-блок

- Заголовок: **"Розы, которые говорят за вас"**
- Подзаголовок: **"Букеты, которые вызывают ВАУ"**
- Кнопка "Выбрать букет" — стиль с золотой обводкой, letter-spacing
- **Под кнопкой** — trust-блок из 3 пунктов в одну строку:
  - 📸 Фото перед отправкой
  - ✓ Свежесть гарантирована
  - 🚚 Доставка от 2 часов
- Убрать "Москва и Московская область" из верхнего тега, заменить на "Премиальная флористика"
- Увеличить отступы (py-20 md:py-32) для воздуха и luxury-ощущения
- Убрать розовый фон-декор за изображением, заменить на тонкую тень

### 2. index.css — золотой акцент + анимации

- Добавить CSS-переменную `--gold: 38 55% 55%` и `--gold-light: 38 60% 85%`
- Добавить `.gold-accent` — тонкая золотая линия-декоратор
- Добавить `@keyframes shimmer` для мягкого свечения на hover

### 3. tailwind.config.ts — цвет gold

- Добавить `gold: { DEFAULT, light }` в extend.colors

### 4. WhyUsSection.tsx — luxury-формулировки

- "Свежесть" → "Только свежие розы" / "Прямые поставки из лучших оранжерей Эквадора"
- "Ручная сборка" → "Авторская флористика" / "Каждый букет — ручная работа мастера"
- "Доставка" → "Бережная доставка" / "Курьер в перчатках, фото перед отправкой"
- "Ежедневно" → "Без выходных" / "Принимаем заказы с 8:00 до 22:00"
- Фон секции — лёгкий градиент cream → rose-light/10

### 5. FeaturedSection.tsx — заголовок

- "Наши букеты" → **"Избранная коллекция"**
- "Коллекция" тег → "Для особенных моментов"

### 6. Header.tsx — минимальное усиление

- Высота h-16 → h-20
- Логотип: добавить тонкий gold-разделитель или увеличить размер шрифта

### 7. ProductCard.tsx — premium hover

- Добавить мягкую розовую тень при hover (`shadow-lg shadow-rose-light/20`)
- Цена — чуть крупнее шрифт
