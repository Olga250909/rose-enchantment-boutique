

## План: Полноценная админ-панель с загрузкой фото и управлением контентом

Текущая админка базовая — нет загрузки фото с устройства, нет управления условиями доставки, нет переключателя наличия товара. Перестроим AdminPage с 3 вкладками и полной функциональностью.

---

### Что будет сделано

**1. Загрузка фото товаров с устройства**
- Заменить текстовое поле "URL изображения" на `<input type="file" accept="image/*">` 
- Фото конвертируется в base64 Data URL и сохраняется в state (работает без бэкенда)
- Превью загруженного фото в форме
- Работает на любом устройстве: компьютер, ноутбук, планшет, смартфон, iPhone (камера + галерея)

**2. Расширенная форма товара**
- Название, описание (textarea), цена, категория (select из существующих + свой вариант), кол-во роз
- Загрузка фото с устройства + возможность вставить URL
- Переключатель "В наличии / Нет в наличии" (Switch)
- Валидация обязательных полей

**3. Вкладка "Заказы" — улучшения**
- Фильтр по статусу (все / новые / в обработке / доставлены)
- Счётчик новых заказов (badge)
- Детальная карточка заказа с товарами и суммой

**4. Новая вкладка "Настройки доставки"**
- Редактирование условий доставки прямо из админки
- Поля: бесплатная доставка от (₽), стоимость доставки, время доставки, стоимость за МКАД, способы оплаты
- Сохранение в StoreContext, отображение на странице /delivery динамически

**5. Обновление StoreContext**
- Добавить `deliverySettings` в state с начальными значениями из текущей страницы доставки
- Добавить `updateDeliverySettings` action

**6. Обновление DeliveryPage**
- Брать данные из `useStore().deliverySettings` вместо хардкода

**7. Адаптивность админки**
- Вкладки: горизонтальный скролл на мобильных
- Форма товара: 1 колонка на мобильных, 2 на десктопе
- Карточки заказов: компактный вид на мобильных

---
