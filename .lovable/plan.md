
Добавить интерактивный конструктор букета «Магия Роз — Конструктор букета» с отдельной страницей и кнопкой на главной.

### Новые файлы

1. **`src/data/constructorOptions.ts`** — конфиг: цвета роз, упаковка, ленты, дополнения с ценами и hex для превью.

2. **`src/pages/ConstructorPage.tsx`** — пошаговый конструктор:
   - **Цвет роз:** красные, белые, розовые, кремовые, микс (250 ₽/шт)
   - **Количество:** пресеты 11/15/25/35/51/75/101 + слайдер
   - **Дополнения (мульти):** эвкалипт +500, лаванда +700, гипсофила +400
   - **Упаковка:** крафт +0, фетр +800, шляпная коробка +1500, премиум-плёнка +500
   - **Лента:** белая атласная / пудровая шёлковая / бордовая бархатная / без ленты
   - **Sticky-превью:** визуальная карточка букета (круг из цветных лепестков на фоне упаковки) + список выбранного + итоговая цена в реальном времени
   - Кнопка **«Добавить в корзину»** → создаёт `Product` с id `custom-{timestamp}`, названием «Авторский букет (N роз)», описанием состава и добавляет через `useCart`

### Изменения

3. **`src/App.tsx`** — роут `/constructor`
4. **`src/components/Header.tsx`** — ссылка «Конструктор» (desktop + mobile)
5. **`src/components/HeroSection.tsx`** — вторая CTA-кнопка «Собери букет сам» рядом с основной (outline, золотая)
6. **`src/components/FeaturedSection.tsx`** — баннер-CTA «Магия Роз — Конструктор букета» в конце секции (опционально, если уместно по структуре)
