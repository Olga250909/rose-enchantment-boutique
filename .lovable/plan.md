Добавить публичную страницу «Оформление праздников» со списком услуг (арки и др.) и формой заявки. Карточки берутся из общего стора `decorServices`, которым уже управляет админка.

### Что появится у посетителей

- Пункт меню **«Оформление»** в шапке (десктоп + мобильное меню), ведёт на `/decor`.
- Страница `/decor`:
  - Заголовок «Оформление праздников» + краткое интро.
  - Фильтр по категориям (Свадьбы / Юбилеи / Корпоративы / Детские праздники) — как на каталоге.
  - Сетка карточек услуг: фото, название, описание, цена «от …», кнопка **«Оставить заявку»**.
  - При клике открывается модальное окно с формой (имя, телефон, дата мероприятия, комментарий).
  - Заявка сохраняется в `orders` (с пометкой «Услуга оформления: …») и видна в админке во вкладке «Заказы».

### Файлы

**Создать**
- `src/pages/DecorPage.tsx` — публичная страница каталога услуг.
- `src/components/DecorCard.tsx` — карточка услуги.
- `src/components/DecorRequestModal.tsx` — модальное окно заявки (имя, телефон, дата, комментарий) с валидацией и форматированием телефона как в `QuickOrderModal`.

**Изменить**
- `src/App.tsx` — добавить маршрут `/decor` → `DecorPage`.
- `src/components/Header.tsx` — добавить ссылку «Оформление» (десктоп и мобильное меню).
- `src/components/Footer.tsx` — добавить ссылку в навигацию (если есть блок навигации).

### Технические детали

- Данные читаются через `useStore().decorServices` — никаких новых сторов не нужно.
- Заявка пишется через существующий `addOrder`: создаётся «виртуальный» товар-услуга `{ id: service.id, name: service.name, price: service.price, image: service.image, category: service.category, description: service.description }` и кладётся в `items` с количеством 1, чтобы заявка корректно отобразилась в `OrdersTab`. В поле `comment` добавляется тип услуги, дата мероприятия и пользовательский комментарий.
- Стилистика — как в `CatalogPage` / `ProductCard` / `QuickOrderModal` (gold-акценты, `font-heading`, `font-body`, `rounded-sm`, без новых токенов).
- Категории берутся из `decorCategories` (экспортирован из `src/data/decorServices.ts`), фильтр «Все» + список категорий, в которых есть услуги.
- Цена выводится как `от {price.toLocaleString('ru-RU')} ₽`.
